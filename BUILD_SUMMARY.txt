   npm run dev

4. START RECORDING:
   Click microphone button and speak

5. SAVE TRANSCRIPTS:
   Edit text and click "Save Transcript"

================================================================================
                            API ENDPOINTS
================================================================================

Authentication (Supabase built-in):
  POST   /auth/signup          - Register
  POST   /auth/signin          - Login
  GET    /auth/me              - Current user
  POST   /auth/signout         - Logout

Transcripts (all require JWT):
  POST   /api/transcripts      - Create transcript
  GET    /api/transcripts      - List user's transcripts
  GET    /api/transcripts/:id  - Get specific transcript
  PATCH  /api/transcripts/:id  - Update transcript
  DELETE /api/transcripts/:id  - Delete transcript

Health Check:
  GET    /health               - Server status
